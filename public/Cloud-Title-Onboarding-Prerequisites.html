Cloud Title Onboarding

Prerequisites Checklist

Please gather the following information before starting the onboarding form. Having everything ready will help you complete the process in about 10-15 minutes.

1 Company Information

  • Company Name REQUIRED
    Your official business name as it should appear in the system
  • Contact Person Name REQUIRED
    Primary point of contact for this onboarding
  • Email Address REQUIRED
    Business email for communications and notifications
  • Phone Number REQUIRED
    Best contact number including area code

2 System Access Credentials

  • Title Capture Login REQUIRED
    Username and password for your Title Capture account
  • GoHighLevel (GHL) Login REQUIRED
    Username and password for your GHL account
  • WordPress Admin Login REQUIRED
    Admin username and password for your WordPress site
Security Tip:

Have your login credentials ready but do not share them via email. You'll enter them securely in the onboarding form.

3 Technical Requirements

  • OpenAI API Key OPTIONAL
    If you have an OpenAI account, have your API key ready. If not, we can help you set one up.
  • WordPress Website URL REQUIRED
    The full URL of your WordPress website (e.g., https://yourcompany.com)
Don't have an OpenAI API Key?

No problem! Simply select "I need help setting this up" in the form and we'll assist you with creating one.

4 Branding Assets

  • Company Logo REQUIRED
    High-resolution logo file (PNG, JPG, or SVG format recommended). Minimum 500x500 pixels.
  • Primary Brand Color REQUIRED
    Your main brand color (hex code if known, e.g., #2563eb, or describe it)
  • Secondary Brand Color REQUIRED
    Your secondary/accent color (hex code if known, or describe it)
Finding Your Brand Colors:

Check your website, business cards, or marketing materials. If you have a brand guidelines document, the hex codes will be listed there.

5 Documents

  • Sample Title Document (PDF) REQUIRED
    A sample PDF of a typical title document you work with. This helps us configure the system correctly.
  • Additional Documents OPTIONAL
    Any other relevant documents, templates, or files you'd like to share

6 Scheduling

  • Calendly Link OPTIONAL
    If you use Calendly for scheduling, have your booking link ready

7 Additional Notes

  • Special Requirements or Questions OPTIONAL
    Think about any specific needs, customizations, or questions you'd like to discuss

Once you have gathered all the required information, you're ready to complete the onboarding form!

Questions? Contact us before starting the form.
